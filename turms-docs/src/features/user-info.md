### 用户资料功能

| **功能**             | **功能描述**                                                 | **相关配置** | 实现版本 |
| :------------------- | :----------------------------------------------------------- | ------------ | -------- |
| 设置用户资料         | 用户设置自己的昵称、介绍、头像URL                            |              | 0.8.0 ✔  |
| 获取用户资料         | 用户查看自己或其他用户的资料                                 |              | 0.8.0 ✔  |
| 设置用户资料访问权限 | 用户可以针对个人的每项资料设置访问权限。访问权限有：所有人可见、好友可见、仅自己可见 |              | 0.8.0 ✔  |
